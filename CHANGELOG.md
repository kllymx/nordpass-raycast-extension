# Changelog

All notable changes to the NordPass Raycast Extension will be documented in this file.

## [1.0.0] - 2024-11-07

### Added
- Initial release of NordPass Raycast Extension
- Search passwords, credit cards, and secure notes
- Fast local search with intelligent caching
- AES-256-GCM encryption for cached data
- Automatic export file detection in `~/.nordpass/` directory
- Secure file permissions (600) for export and cache files
- Auto-refresh when export file changes
- Copy to clipboard functionality for all credential types
- Last updated timestamp display
- Clear cache functionality
- Comprehensive security documentation
- Keyboard shortcuts for common actions

### Security
- Encrypted cache storage using AES-256-GCM
- System-specific encryption keys
- Automatic secure file permissions
- Local-only data storage (no external servers)
- Read-only access to export files

### Documentation
- Complete setup guide
- Security best practices
- Troubleshooting guide
- Keyboard shortcuts reference
